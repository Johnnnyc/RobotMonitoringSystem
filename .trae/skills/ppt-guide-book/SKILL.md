# PPT指导书生成器

## 功能

此技能专门用于创建各种指导书类型的PPT，包括但不限于：
- 维修指导书
- 操作原理说明书
- 技术培训手册
- 设备使用指南
- 故障排除手册

## 使用场景

当用户需要：
- 制作结构统一的技术指导类PPT
- 基于标准模板生成维修或操作指南
- 快速创建包含步骤说明的培训材料
- 确保指导书内容的一致性和专业性

## 标准模板结构

指导书类型PPT的标准结构包括：

1. **封面**
   - 标题
   - 副标题
   - 版本信息
   - 日期
   - 公司/部门标识

2. **目录**
   - 章节列表
   - 页码

3. **概述**
   - 文档目的
   - 适用范围
   - 术语定义

4. **详细内容**
   - 结构清晰的章节
   - 步骤化说明
   - 图示和流程图
   - 注意事项和警告

5. **故障排除**
   - 常见问题
   - 解决方案

6. **附录**
   - 参考资料
   - 相关文档
   - 联系信息

## 使用方法

1. 调用此技能
2. 提供以下信息：
   - 指导书类型（维修、操作、培训等）
   - 主题内容
   - 目标受众
   - 特殊要求（如有）
3. 技能将基于标准模板生成PPT结构和内容建议

## 示例

### 维修指导书示例

**主题**：工业机器人定期维护指南

**结构**：
1. 封面
2. 目录
3. 概述
4. 安全注意事项
5. 维护周期表
6. 详细维护步骤
   - 机械部分
   - 电气部分
   - 控制系统
7. 常见故障排除
8. 维护记录表格
9. 附录

### 操作原理说明书示例

**主题**：自动化生产线工作原理

**结构**：
1. 封面
2. 目录
3. 概述
4. 系统组成
5. 工作流程
6. 关键组件原理
7. 操作步骤
8. 监控与调整
9. 附录

## 注意事项

- 确保内容准确无误，特别是技术参数和操作步骤
- 根据目标受众调整专业术语的使用
- 包含足够的图示和说明，提高可读性
- 遵循公司或行业的标准格式要求
